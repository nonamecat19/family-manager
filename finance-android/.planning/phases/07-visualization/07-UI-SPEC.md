## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Pie chart section gaps (sectionsSpace: 2-4), icon-label gaps |
| sm | 8px | Chip gaps in month selector, compact vertical spacing |
| md | 16px | Screen horizontal padding, section padding, card internal padding |
| lg | 24px | Space between chart and legend, space between chart sections |
| xl | 32px | Space between month selector and chart content |
| 2xl | 48px | Not used in this phase |
| 3xl | 64px | Not used in this phase |

Exceptions: Pie chart radius at 100px and center space at 40px (fl_chart geometry, not layout spacing). Bar width at 12px (chart element sizing).

**Source:** Existing codebase uses `EdgeInsets.all(16)`, `SizedBox(height: 8)`, `EdgeInsets.symmetric(horizontal: 16, vertical: 8)` consistently.

## Typography

| Role | Size | Weight | Line Height | Usage in This Phase |
|------|------|--------|-------------|---------------------|
| Body (bodyMedium) | 14px | 400 (regular) | 1.5 | Category names in legend, summary list descriptions |
| Label (bodySmall) | 12px | 400 (regular) | 1.4 | Chart axis labels, day numbers on bar chart, pie chart section labels |
| Heading (titleMedium) | 16px | 500 (medium) | 1.3 | Month selector label ("March 2026"), section headings |
| Display (headlineMedium) | 28px | 400 (regular) | 1.2 | Monthly total amount ("$1,250.00") at top of summary |

**Source:** Existing screens use `theme.textTheme.titleMedium`, `theme.textTheme.bodyMedium`, `theme.textTheme.bodySmall` from Material 3 defaults.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `ColorScheme.surface` (generated from teal seed) | Screen background, chart background area |
| Secondary (30%) | `ColorScheme.surfaceContainerLow` | Summary list cards, legend container background |
| Accent (10%) | `ColorScheme.primary` (teal-derived) | Month selector arrows, active bar chart fill, selected state indicators |
| Destructive | not applicable | No destructive actions in this phase |

Accent reserved for: month navigation arrow icons, bar chart bar fill color (when not showing per-category breakdown), loading indicator, active/selected month highlight.

**Chart-specific color rules:**
- Pie chart sections: use each category's user-assigned color from `category.color` field via `parseHexColor()`. Never hardcode chart colors.
- Bar chart bars: use `ColorScheme.primary` (teal) for daily total bars.
- Pie chart section labels: white (`Colors.white`) with `FontWeight.bold` at 12px for contrast on colored sections.

**Source:** RESEARCH.md anti-pattern "Hardcoded chart colors -- categories already have user-assigned colors, use them directly."

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | None (this phase is read-only visualization, no primary action) |
| Empty state heading | "No spending data" |
| Empty state body | "Start logging expenses to see your spending breakdown here." |
| Empty state icon | Icons.pie_chart_outline, size 48, ColorScheme.onSurfaceVariant |
| Error state | "Couldn't load spending summary. Pull down to try again." |
| Loading state | CircularProgressIndicator centered (matches existing pattern) |
| Month selector label | DateFormat.yMMMM() output, e.g. "March 2026" |
| Summary total label | "Total Spent" |
| Category count label | "{count} expense" (singular) / "{count} expenses" (plural) |
| Bar chart Y-axis format | Abbreviated: "$0", "$500", "$1K", "$1.5K", "$2K" |
| Pie chart percentage format | "{percent}%" with no decimal for >= 1%, "< 1%" for values under 1% |

## Interaction States

| State | Visual Treatment |
|-------|-----------------|
| Loading (initial) | Centered CircularProgressIndicator (matches HistoryScreen) |
| Loaded (with data) | All three visualizations render with data |
| Loaded (no data for month) | Empty state: icon + heading + body copy (no charts rendered) |
| Error | Centered error text with retry instruction |
| Month navigation | Instant state transition to loading, then loaded |
| Pie section touch | Touched section radius expands from 100 to 110, 300ms animation |
| Bar touch | Tooltip appears above touched bar showing formatted amount |
| Future month blocked | Right chevron disabled (grayed out) when on current month |
