4. Create `app/lib/features/family/data/family_repository.dart`:
FamilyRepository class with Dio ref (same pattern as CategoryRepository/ExpenseRepository). Methods:
- `Future<Map<String, dynamic>> getMyFamily()` — GET /api/v1/families/me, returns raw JSON map with family, members, invitations
- `Future<Family> createFamily(String name)` — POST /api/v1/families with `{"name": name}`, returns Family.fromJson
- `Future<void> deleteFamily()` — DELETE /api/v1/families/me
- `Future<void> removeMember(String userId)` — DELETE /api/v1/families/me/members/$userId
- `Future<void> leaveFamily()` — POST /api/v1/families/me/leave
- `Future<String> createInvitation()` — POST /api/v1/families/me/invitations, returns raw token string from `response.data['token']`
- `Future<void> revokeInvitation(String id)` — DELETE /api/v1/families/me/invitations/$id
- `Future<Map<String, dynamic>> getInvitationInfo(String token)` — GET /api/v1/invitations/$token, returns JSON with family_name
- `Future<void> acceptInvitation(String token)` — POST /api/v1/invitations/accept with `{"token": token}`

Provider: `final familyRepositoryProvider = Provider<FamilyRepository>((ref) => FamilyRepository(ref.watch(dioProvider)));`

5. Create `app/lib/features/family/domain/family_state.dart`:
```dart
sealed class FamilyState {
  const FamilyState();
}
class FamilyInitial extends FamilyState { const FamilyInitial(); }
class FamilyLoading extends FamilyState { const FamilyLoading(); }
class FamilyLoaded extends FamilyState {
  final Family family;
  final List<FamilyMember> members;
  final List<FamilyInvitation> invitations;
  const FamilyLoaded(this.family, this.members, this.invitations);
}
class NoFamily extends FamilyState { const NoFamily(); }
class FamilyError extends FamilyState {
  final String message;
  const FamilyError(this.message);
}
```

6. Create `app/lib/features/family/domain/family_notifier.dart`:
FamilyNotifier extends StateNotifier<FamilyState> with initial state FamilyInitial.
Methods:
- `loadFamily()` — calls repository.getMyFamily(), parses response into Family + List<FamilyMember> + List<FamilyInvitation>, sets FamilyLoaded. On 404 DioException, sets NoFamily. On other errors, sets FamilyError.
- `createFamily(String name)` — calls repository.createFamily(name), then loadFamily()
- `deleteFamily()` — calls repository.deleteFamily(), sets NoFamily
- `removeMember(String userId)` — calls repository.removeMember(userId), then loadFamily()
- `leaveFamily()` — calls repository.leaveFamily(), sets NoFamily
- `createInvitation()` — calls repository.createInvitation(), returns raw token string, then loadFamily() to refresh invitation list
- `revokeInvitation(String id)` — calls repository.revokeInvitation(id), then loadFamily()
- `acceptInvitation(String token)` — calls repository.acceptInvitation(token), then loadFamily()

Providers:
```dart
final familyStateProvider = StateNotifierProvider<FamilyNotifier, FamilyState>((ref) {
  return FamilyNotifier(ref.watch(familyRepositoryProvider));
```
