```sql
-- +goose Up
CREATE TABLE families (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name TEXT NOT NULL,
    admin_user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE TABLE family_members (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    family_id UUID NOT NULL REFERENCES families(id) ON DELETE CASCADE,
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    role TEXT NOT NULL DEFAULT 'member' CHECK (role IN ('admin', 'member')),
    joined_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    UNIQUE(user_id)
);

CREATE INDEX idx_family_members_family_id ON family_members(family_id);
CREATE INDEX idx_family_members_user_id ON family_members(user_id);

CREATE TABLE family_invitations (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    family_id UUID NOT NULL REFERENCES families(id) ON DELETE CASCADE,
    inviter_user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    token_hash TEXT NOT NULL UNIQUE,
    status TEXT NOT NULL DEFAULT 'pending' CHECK (status IN ('pending', 'accepted', 'revoked', 'expired')),
    expires_at TIMESTAMPTZ NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_family_invitations_token_hash ON family_invitations(token_hash);
CREATE INDEX idx_family_invitations_family_id ON family_invitations(family_id);

-- +goose Down
DROP TABLE IF EXISTS family_invitations;
DROP TABLE IF EXISTS family_members;
DROP TABLE IF EXISTS families;
```

- `RevokeInvitation(ctx context.Context, invitationID, familyID string) (int64, error)`
- `GetPendingInvitations(ctx context.Context, familyID string) ([]sqlc.GetPendingInvitationsRow, error)`

Define `FamilyHandler` struct with `db FamilyDB` field and `NewFamilyHandler(db FamilyDB) *FamilyHandler` constructor.

Implement these Gin handlers on FamilyHandler:

- `CreateFamily` — POST /families: Parse `{"name": "..."}`, validate non-empty name, check user not already in family (GetFamilyByUserID returns pgx.ErrNoRows = OK), create family, add creator as admin member. Return 201 with family JSON.
- `GetMyFamily` — GET /families/me: GetFamilyByUserID, if not found return 404 `{"error":"no family"}`. If found, also fetch members and pending invitations (admin only for invitations). Return 200 with `{"family": {...}, "members": [...], "invitations": [...]}`.
- `DeleteMyFamily` — DELETE /families/me: Verify user is admin (check admin_user_id), delete family (CASCADE removes members+invitations). Return 200.
- `RemoveMember` — DELETE /families/me/members/:userId: Verify requester is admin, call RemoveFamilyMember, check rows affected. Return 200.
- `LeaveFamily` — POST /families/me/leave: Get family, verify user is NOT admin (admin must use delete), remove member. Return 200.
- `CreateInvitation` — POST /families/me/invitations: Verify user is admin, check member count < 10, generate 32-byte crypto/rand token, hex encode, SHA-256 hash, store hash in DB with 7-day expiry. Return 201 with `{"token": "raw_hex_token", "expires_at": "..."}`.
- `RevokeInvitation` — DELETE /families/me/invitations/:id: Verify admin, revoke. Return 200.
- `GetInvitationInfo` — GET /invitations/:token: SHA-256 hash the token, look up by hash, return family name and expiry (public-ish preview). Return 200 or 404.
- `AcceptInvitation` — POST /invitations/accept: Parse `{"token": "..."}`, hash token, look up invitation, check not expired, check user not already in family, check family member count < 10, add member, mark invitation accepted. Return 200 with family JSON.

Token generation: `crypto/rand.Read(make([]byte, 32))` then `hex.EncodeToString()`. Hashing: `sha256.Sum256([]byte(rawToken))` then `hex.EncodeToString(hash[:])`. This mirrors the refresh token pattern in auth.go.
